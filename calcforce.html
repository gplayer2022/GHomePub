N （ニュートン）と kgf （重量キログラム）

kgf （重量キログラムあるいはキログラム重）とは、 1kg の質量を持つものが、およそ 9.8m/s2 の重力加速度のもとで受ける重力の大きさです。物理基礎では、力の単位として N （ニュートン）を使用します。

計算室

計算

  • 力 : N （指定可能範囲 : [0.0, 9999.9] ）

  • 重量キログラム : kgf （指定可能範囲 : [0.0, 9999.9] ）

閑話休題

1N は 0.102kgf になります。およそ 100g 強の物を持ちあげたときに感じる重さです。

N という単位に慣れたければ、 コンビニのおにぎりを持ちあげるたびに、掌に 1N の力を自覚することです。あなたの掌がおにぎりを押しあげるとき、おにぎりもあなたの掌を押しているのです。
