フレックスボックス練習帳

このページでは CSS のフレックスボックスをガンツやクレアと一緒に学んでいきます。

ガンツはドワーフの万屋よろずや経営者です。最近、世間では業務の IT 化がすっかり進んでいるのを知って、年寄りの手習いで自分でもシステムをいじり始めています。

メイドのクレアはガンツに雇われた普通の人間です。ときどき少し口が悪くなることもあります。 IT に詳しく、ガンツの経営の IT 化を補助しています。

01. ドワーフ・ガンツのフレックスコンテナ

クレアさんよぅ、 CSS を使ってフレックスアイテムたちを横並びにしたいんじゃが、うまくできないものかな？

フレックスボックスを使えば簡単に横並びにできますよ。

解説 01

ここをクリックすれば解説が表示されます。

{ display: flex;} を使うと、フレックスアイテムたちを一次元の横方向に並べることができます。

div.target {
  display: flex;
}

これだけ？

はい。これだけです。デフォルトで左から右への横方向に並びますが、 flex-direction を使えば、右から左、上から下、下から上方向も指定できます。

div.target {
  display: flex;
  flex-direction: row; /* 左から右（デフォルト） */
  flex-direction: row-reverse; /* 右から左 */
  flex-direction: column; /* 上から下 */
  flex-direction: column-reverse; /* 下から上 */
}

わざわざ上から下へとか並べて意味あるんかいな？ そもそも flex なんか使わなければ嫌でも上から下に並ぶじゃろ？

もちろん意味はあります。スマホで見るときみたいに狭い横幅に対応させたいときと、パソコンで見るときみたいに広い横幅に対応させたいときとで表示を切り替えるような、レスポンシブデザインを実装する際なんかには重宝します。 

/* スマホファーストで */
div.target {
  display: flex;
  flex-direction: column; /* 上から下 */
}
/* PC 表示の場合 */
@media (min-width: 884px) {
  div.target {
    flex-direction: row; /* 左から右 */
  }
}

なるほど。

ちなみに flex 指定された領域を「フレックスコンテナ」と言います。また、フレックスコンテナ内の子要素たちを「フレックスアイテム」と言います。

設問 01

{ display: flex;} 指定だけで横並びになります。右から左に並べてください。

.css-panel-answer {
  display: flex;

}

解答例 01

解答例です。設問欄で他の向きも試してみるとよいでしょう。

.css-panel-answer {
  display: flex; /* 設定済 */
  flex-direction: row-reverse;
}

02. ドワーフ・ガンツの justify-content

もう少しいい感じに真ん中に設定できたりしないのかな？

justify-content を使えば flex-direction の方向を基準にして、簡単に中央配置させられます。

解説 02

ここをクリックすれば解説が表示されます。

justify-content を使うと、一次元に並んだフレックスアイテムたちの配置を変更することができます。

.css-panel-answer {
  justify-content: flex-start; /* 先頭側から配置（デフォルト） */
}

このとき、注意して欲しいのは、 flex-direction の指定次第で先頭の位置が変わるということです。

デフォルトの row だったら左から右で、例えば row-reverse だったら同じように flex-start にしていても、右から左に並ぶんじゃな。

そうですね。終端側に押し込むには次のようにします。

.css-panel-answer {
  justify-content: flex-end; /* 終端側から配置 */
}

他には例えば均等な間隔で並べるためのこんなプロパティたちがあります。

.css-panel-answer {
  /* まず両端に配置して、その間に要素を等間隔で埋める */
  /* 例 : |A------B------C| */
  justify-content: space-between;
  /* 各要素の両端のマージンが全て等しい */
  /* 例 : |--A----B----C--| */
  justify-content: space-around;
  /* 全ての余白が等しい */
  /* 例 : |---A---B---C---| */
  justify-content: space-evenly;
}

ほう。 3 つもあるんじゃな。

中央配置は次の通りです。

.css-panel-answer {
  justify-content: center; /* 中央に配置 */
}

設問 02

マンホールたちを中央配置してください。

.css-panel-answer {
  display: flex;

}

解答例 02

解答例です。

.css-panel-answer {
  display: flex; /* 設定済 */
  justify-content: center;
}

03. ドワーフ・ガンツの応用

flex-direction と justify-content を組み合わせただけで、いろいろできそうじゃな？

そうですね。

解説 03

ここをクリックすれば解説が表示されます。

flex-direction を使うことで、左から右・右から左・上から下・下から上の指定ができました。

.css-panel-answer {
  flex-direction: row; /* 左から右（デフォルト） */
  flex-direction: row-reverse; /* 右から左 */
  flex-direction: column; /* 上から下 */
  flex-direction: column-reverse; /* 下から上 */
}

justify-content を使うことで、開始端寄せ・終了端寄せ・中央寄せができる他、均等割り付けができます。

均等割り付けは 3 種類できるんじゃったな。

そうです。

.css-panel-answer {
  justify-content: flex-start; /* 開始端から配置（デフォルト） */
  justify-content: flex-end; /* 終端から配置 */
  justify-content: center; /* 中央寄せ */
  /* 均等割り付け */
  justify-content: space-between; /* 例 : |A------B------C| */
  justify-content: space-around;  /* 例 : |--A----B----C--| */
  justify-content: space-evenly;  /* 例 : |---A---B---C---| */
}

組み合わせれば、これだけでも 24 通りできるんじゃな。

はい。ただ、今のところ、左上寄りのレイアウトばかりです。

設問 03

マンホールに蓋をしてください。

.css-panel-answer {
  display: flex;

}

解答例 03

解答例です。

.css-panel-answer {
  display: flex; /* 設定済 */
  flex-direction: column-reverse;
  justify-content: space-evenly;
}

04. ドワーフ・ガンツの交差軸

flex-direction と justify-content を使うだけだと、全体的にフレックスアイテムが左上によってしまうようじゃが、もっと右下の領域も活用できないものかな？

align-items を使って「交差軸」を指定すれば、もっと柔軟にフレックスコンテナを活用できます。

交差軸とな？

解説 04

ここをクリックすれば解説が表示されます。

すでに学んだ通り、 flex-direction を使うことで、左から右・右から左・上から下・下から上の指定ができました。

.css-panel-answer {
  flex-direction: row; /* 左から右（デフォルト） */
  flex-direction: row-reverse; /* 右から左 */
  flex-direction: column; /* 上から下 */
  flex-direction: column-reverse; /* 下から上 */
}

flex-direction による指定でフレックスアイテムたちが並ぶ直線を「主軸」といいます。

デフォルトの row なら左から右への直線が主軸で、一方 column 指定だったら上から下への直線が主軸なんじゃな？

そうです。それに対して、主軸と垂直方向の軸を「交差軸」と言います。

つまり、例えば flex-direction の指定がデフォルトの row なら、上下方向が交差軸なんじゃな？

そうです。交差軸は align-items で指定します。デフォルトは stretch です。

.css-panel-answer {
  align-items: stretch; /* 交差軸いっぱいに伸ばす */
}

実はこれは今までやってきたマンホールみたいに、画像では何が起こっているのかよく分かりません。テキストを使ってこんな風に指定してみると、……

div.target {
  display: flex;
  /* フレックスアイテム間の隙間 */
  gap: 6px;
  /* 交差軸の設定 */
  align-items: stretch;
}

/* フレックスアイテムたちを 1/4 幅に */
div.target p {
  width: 25%;
}

こんな風になります。

なるほど、一番長いテキストに合わせて、フレックスコンテナの交差軸いっぱいに広がるんじゃな。

はい。ただしこれは、フレックスアイテムたちの幅を指定しているからです。フレックスアイテムたちの幅を指定しないとこんな感じになっちゃいます。

すべてのフレックスアイテムたちの文字列が交差軸方向いっぱい広がる感じで、幅が調整されてしまうんじゃな。

はい。続けて、位置を調整するタイプの値はこんな感じです。

div.target {
  align-items: flex-start; /* 開始端に合わせる */
  align-items: flex-end;   /* 終了端に合わせる */
  align-items: center; /* 中央に合わせる */
}

これもマンホールみたいな画像だけでは分かりにくいところがあるので、テキストの例で示しておきますね。まずは flex-start の例です。マンホール画像では flex-start を指定しても何も変化が感じられないはずです。

いっぱいに広がらなくなったな。

flex-end の例です。

center の例です。

位置調整タイプの値だと、いずれも交差軸いっぱいに広がらなくなるんじゃな。

ちなみに、特定の要素だけを交差軸方向に位置調整するには、該当の要素に対して直接 align-self を指定します。

div.target {
  align-items: flex-start; /* 開始端に合わせる */
}
.blue-item {
  align-self: flex-end; /* 青い要素のみ終了端に合わせる */
}

こんな感じになります。

なるほど。ところで align を使ったプロパティというと、テキストの位置を中央寄せにしたり左寄せにしたりする text-align がすぐに思い浮かぶ。 text-align の方は「なにをどうするか」みたいな書き方なのに、 align-items の方はメソッド名によくある命令文みたいな書き方になってるんじゃな。

CSS 増築の歴史がそのまま現れてるんでしょうね。 text-align は 2000 年以前からある古い CSS なのに対して、フレックスボックスが登場したのは 2010 年代です。少なくともフレックスボックス関連のプロパティとしては、 justify-content なども命令文みたいな書き方になってますし、フレックスボックス内では統一されてはいる気はします。

まあ、こればっかりは受け入れるしかない感じじゃな。……フレックスボックスがなかったころは、横並びとかどうしてたんじゃろうな？

フロート・レイアウトというものを使ってたみたいですよ。かなり癖があってめんどくさかったみたいです。私はピチピチギャルなので、そんな昔のことは分かりませんけど。

そ、そうじゃな……。

設問 04

マンホールの蓋を交差軸の終端側に移動してください。

.css-panel-answer {
  display: flex;

}

解答例 04

解答例です。

.css-panel-answer {
  display: flex; /* 設定済 */
  align-items: flex-end;
}

05. ドワーフ・ガンツの主軸と交差軸

主軸と交差軸の設定を組み合わせれば、フレックスアイテムたちを色んな風に配置できそうじゃな。

はい。復習がてらに確認しておきましょう。

解説 05

ここをクリックすれば解説が表示されます。

まず flex-direction を使うと、主軸方向を指定できました。

div.target {
  flex-direction: row; /* 左から右（デフォルト） */
  flex-direction: row-reverse; /* 右から左 */
  flex-direction: column; /* 上から下 */
  flex-direction: column-reverse; /* 下から上 */
}

ふむ。次に勉強した justify-content を使うと、主軸上で先頭側か終端側か中央かだけでなく、均等配置もできるんじゃったな。

はい。

div.target {
  justify-content: flex-start; /* 開始端から配置（デフォルト） */
  justify-content: flex-end; /* 終端から配置 */
  justify-content: center; /* 中央寄せ */
  /* 均等割り付け */
  justify-content: space-between; /* 例 : |A------B------C| */
  justify-content: space-around;  /* 例 : |--A----B----C--| */
  justify-content: space-evenly;  /* 例 : |---A---B---C---| */
}

交差軸は align-items で位置調整するんじゃな。

はい。

div.target {
  align-items: flex-start; /* 開始端に合わせる */
  align-items: flex-end;   /* 終了端に合わせる */
  align-items: center; /* 中央に合わせる */
}

デフォルトは stretch で交差軸いっぱいに伸びるんじゃったな。

はい。

設問 05

マンホールに蓋をしてください。

.css-panel-answer {
  display: flex;

}

解答例 05

解答例です。

.css-panel-answer {
  display: flex; /* 設定済 */
  flex-direction: column-reverse;
  align-items: center;
  justify-content: space-around;
}

06. ドワーフ・ガンツの改行

横並びはいいんじゃが、編集中にフレックスコンテナに要素を追加したくなって、いい感じに改行してくれないかなぁと思うこともあるじゃろ？

そうなんですか？

そうなんじゃ。上手い具合にチャチャッとできないかな？

flex-wrap プロパティを使うだけで、上手い具合にチャチャッとできますよ。

解説 06

ここをクリックすれば解説が表示されます。

特にフレックスコンテナ内のフレックスアイテムたちが画像データの場合、幅や高さが十分でないと、勝手に画像が縮んだりフレックスコンテナからはみ出したりします。改行を扱うプロパティは flex-wrap で、デフォルト値は nowrap になっています。

div.target {
  flex-wrap: nowrap; /* 改行なし */
}

改行させたい場合は nowrap の代わりに wrap を指定します。

div.target {
  flex-wrap: wrap;
}

これだけ？

はい。上手い具合にチャチャッとできたでしょ？

おう。

ちなみに flex-wrap と flex-direction は、プロパティ flex-flow にまとめて設定することができます。

div.target {
  /* 別々に設定 */
  flex-direction: row-reverse;
  flex-wrap: wrap;
  /* まとめて設定 */
  flex-flow: row-reverse wrap;
}

まとめた方がよいのかな？

別に無理にまとめる必要はありません。

設問 06

改行してマンホールに蓋をしてください。

.css-panel-answer {
  display: flex;

}

解答例 06

解答例です。

.css-panel-answer {
  display: flex; /* 設定済 */
  flex-wrap: wrap;
}

07. ドワーフ・ガンツのフレックスボックスまとめ

フレックスボックス関連のいろんなプロパティが出てきたな。全部使えばいろいろできそうじゃ。

小手試しといきますか？

おう！

解説 07

ここをクリックすれば解説が表示されます。

復習を兼ねて簡単にまとめておきましょう。

まずは flex-direction じゃな。主軸の方向を決める。

div.target {
  flex-direction: row; /* 左から右（デフォルト） */
  flex-direction: row-reverse; /* 右から左 */
  flex-direction: column; /* 上から下 */
  flex-direction: column-reverse; /* 下から上 */
}

続いて justify-content じゃ。配置や均等割り付けを決める。

div.target {
  justify-content: flex-start; /* 開始端から配置（デフォルト） */
  justify-content: flex-end; /* 終端から配置 */
  justify-content: center; /* 中央寄せ */
  /* 均等割り付け */
  justify-content: space-between; /* 例 : |A------B------C| */
  justify-content: space-around;  /* 例 : |--A----B----C--| */
  justify-content: space-evenly;  /* 例 : |---A---B---C---| */
}

align-items で交差軸を調整する。

div.target {
  align-items: stretch; /* 交差幅いっぱいに伸びる（デフォルト） */
  align-items: flex-start; /* 開始端に合わせる */
  align-items: flex-end;   /* 終了端に合わせる */
  align-items: center; /* 中央に合わせる */
}

flex-wrap で改行の有無を切り替える。

div.target {
  flex-wrap: nowrap; /* 改行なし（デフォルト） */
  flex-wrap: wrap; /* 改行あり */
}

どうじゃ？

ばっちりですね。

設問 07

マンホールに蓋をしてください。

.css-panel-answer {
  display: flex;

}

解答例 07

解答例です。これが最後の問題でした。お疲れ様です。

.css-panel-answer {
  display: flex; /* 設定済 */
  flex-direction: column-reverse;
  justify-content: space-between;
  align-items: flex-end;
  flex-wrap: wrap;
}
